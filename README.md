# Infer [![Build Status](https://travis-ci.org/facebook/infer.svg?branch=master)](https://travis-ci.org/facebook/infer)

[Infer](http://fbinfer.com/) is a static analysis tool for Java,
C++, Objective-C, and C. Infer is written in [OCaml](https://ocaml.org/).


## Installation

Read [Getting
Started](http://fbinfer.com/docs/getting-started.html) page to install packaged versions of Infer. See [INSTALL.md](./INSTALL.md) to build Infer from source.
